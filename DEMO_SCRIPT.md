# 现场演示脚本（5-8 分钟）

## 0) 启动（30 秒）

```powershell
.\start-demo.ps1 -OpenBrowser
```

开场台词建议：

> 这是我做的一个可直接展示的个人作品集成品，它不仅展示作品，也展示我和 Claude Code 的协同方式。

## 1) 首页（1 分钟）

- 讲首页 Hero：项目定位与协同方法
- 下拉到精选作品：说明数据驱动与模板一致性
- 强调“不是单页模板，而是完整内容链路”

## 2) 作品页（1.5 分钟）

- 进入 `works.html`
- 点击分类筛选（摄影叙事 / 平面设计 / 图文创意）
- 强调 12 个案例可扩展，结构统一，替换成本低

## 3) 详情页（1.5 分钟）

- 打开任意案例详情
- 展示细节图、长文本叙事、上下篇导航
- 手动改一个错误链接示例：`work.html?id=not-found`
- 说明系统有兜底状态，不会白屏

## 4) 协同页（1.5 分钟）

- 进入 `collaboration.html`
- 讲 KPI、阶段流程、里程碑证据
- 说明“Claude 主实现，Codex 做双层评审门禁”

## 5) 收束（30 秒）

收尾台词建议：

> 这个站点可以直接作为展示资产，同时还能一键转成短视频素材，内容与分发在同一个流程里完成。
