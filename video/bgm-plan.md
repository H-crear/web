# BGM 节拍方案（30 秒）

## 推荐参数

- 风格：电子叙事 / 科技氛围
- BPM：126-132
- 结构：Intro 4 拍 -> Build 16 拍 -> Drop 16 拍 -> Outro 8 拍

## 卡点建议

1. `00:00` 首个鼓点：项目冲突开场。
2. `00:05` 转折点：出现“协同分工”字卡。
3. `00:18` Drop 点：切到成品站点展示。
4. `00:26` 收束段：出现“网站 + 视频包”并降低能量。
5. `00:30` 结束拍：Logo/标题定格结束。

## 混音建议

- BGM 音量：-14 LUFS 左右
- 字幕风格为主，不放口播时避免过强低频掩盖画面信息
- 切镜转场可用 Whoosh，音量控制在主音乐下方 6-8dB
