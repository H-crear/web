# 30 秒分镜脚本（故事叙事）

## 结构

- 0-5s：冲突（一个人做项目，容易慢且乱）
- 5-18s：协同（Codex 拆解 + Claude Code 实现）
- 18-26s：成果（网站多页和案例切换）
- 26-30s：收束（协同价值结论）

## 逐秒镜头

| 时间 | 画面 | 屏幕字幕 | 节奏点 |
|---|---|---|---|
| 0.0-1.5s | 黑底快速切到空白项目目录 | 「接到一个展示项目」 | 开场钩子，首个鼓点切入 |
| 1.5-3.0s | 终端 + 待办草图闪切 | 「时间紧，要求还高」 | 1 次故障闪白 |
| 3.0-5.0s | 画面定格“一个人容易卡住” | 「单打独斗很容易卡住」 | 低频下潜，准备转折 |
| 5.0-7.5s | 出现双角色标题：Codex / Claude | 「于是我让 AI 协同分工」 | 主节奏进入 |
| 7.5-10.5s | 网站样式代码与页面骨架切换 | 「Codex把关方向，Claude主实现」 | 2 次节拍切镜 |
| 10.5-13.5s | 数据文件与作品卡片渲染 | 「12个案例，自动渲染详情页」 | 节拍密度提升 |
| 13.5-16.0s | 协同页时间线滑动展示 | 「每一步都有质量门禁」 | 镜头推近 |
| 16.0-18.0s | 启动脚本执行画面 | 「一键启动，现场可演示」 | Drop 前停顿 |
| 18.0-20.5s | 首页 Hero 全屏展示 | 「成品出来了」 | Drop 点切换 |
| 20.5-22.5s | 作品列表筛选与悬浮动效 | 「多页结构 + 高动效」 | 连续两拍切镜 |
| 22.5-24.5s | 详情页细节图和上下篇跳转 | 「内容、细节、链路全打通」 | 节奏持续 |
| 24.5-26.0s | 协同页 KPI 和里程碑 | 「流程也能成为作品的一部分」 | 轻推镜头 |
| 26.0-28.0s | 网站 + 视频素材文件夹同屏 | 「网站成品 + 视频素材包」 | 进入收束段 |
| 28.0-30.0s | Logo / 标题收尾 | 「这就是 Codex × Claude 的协同效率」 | 尾奏结束 |

## 拍摄素材建议

1. 终端素材：执行 `start-demo.ps1`、切换目录、展示文件结构。
2. 网站素材：首页、作品页、详情页、协同页各录 3-5 秒。
3. 证据素材：`video/` 文件夹内分镜、SRT、剪辑指南文件滚动展示。
